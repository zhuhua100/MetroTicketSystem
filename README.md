# 模拟地铁自动售票系统 (Metro Ticket System)

## 项目简介
基于 C11 + EasyX 图形库开发的 Windows 图形化地铁售票系统，
模拟真实地铁站自动售票机完整业务流程。

## 功能模块
1. **欢迎界面** - 程序启动首界面
2. **系统说明** - 功能介绍与操作指南
3. **主菜单** - 系统核心导航
4. **地图查询** - 地铁线路图可视化
5. **站点管理** - 站点数据读取与查询
6. **票价计算** - 自动计算票价
7. **购票模块** - 选择起终点购票
8. **支付找零** - 现金支付与自动找零
9. **文件管理** - TXT/BIN双格式存储
10. **历史记录** - 购票订单查看
11. **复杂查询** - 多条件组合查询
12. **系统测试** - 集成测试与断言测试

## 开发环境
- 开发语言: C11
- 图形库: EasyX
- IDE: Visual Studio 2022
- 操作系统: Windows 10/11

## 项目结构
```
MetroTicketSystem/
├── docs/          # 文档
├── include/       # 头文件
├── src/           # 源文件
├── resource/      # 资源文件
├── data/          # 数据文件
├── test/          # 测试
└── record/        # AI聊天记录
```

## 编译方法
```bash
# 使用 MSVC 编译
cl /Fe:MetroTicketSystem.exe /I include src/*.c
```
