# 模拟地铁自动售票系统 - 开题报告

## 一、项目信息

| 项目 | 内容 |
|:----|:-----|
| 项目名称 | 模拟地铁自动售票系统（Metro Ticket System） |
| 开发语言 | C11 |
| 图形库 | EasyX |
| 开发环境 | Visual Studio 2022 |
| 操作系统 | Windows 10/11 |

## 二、项目背景

随着城市轨道交通的快速发展，地铁自动售票系统已成为城市公共交通的重要组成部分。本课程设计旨在通过模拟地铁自动售票系统的开发，综合运用C语言程序设计知识，包括结构体、文件操作、图形界面编程等技术，实现一个具有完整业务流程的地铁售票模拟系统。

## 三、项目目标

1. 开发一个具有图形界面的地铁自动售票系统
2. 实现欢迎界面、系统说明、地图查询、自动售票、自动找零等功能
3. 支持TXT和BIN双格式文件存储
4. 实现多条件组合查询
5. 完成软件测试和Bug修复
6. 进行AI方案对比实验

## 四、系统功能模块

系统共分为13个模块：

1. **Welcome** - 欢迎界面模块
2. **Instruction** - 系统说明模块
3. **Menu** - 主菜单模块
4. **Map** - 地图查询模块
5. **Station** - 站点管理模块
6. **Ticket** - 购票模块
7. **Payment** - 支付找零模块
8. **Price** - 票价计算模块
9. **FileManager** - 文件管理模块
10. **History** - 历史记录模块
11. **Search** - 多条件查询模块
12. **Test** - 系统测试模块
13. **Exit** - 退出模块

## 五、技术方案

- **开发语言**: C11
- **图形界面**: EasyX图形库
- **数据存储**: TXT文本 + BIN二进制双格式
- **数据结构**: 结构体（Station、Line、Ticket、Order）
- **编码**: UTF-8
