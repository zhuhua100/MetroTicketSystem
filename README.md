# 模拟地铁自动售票系统 (Metro Ticket System)

## 版本说明

本仓库包含两个版本：

### v1.0 - 控制台版 ✅（已完成）
- 基于 C11 + 控制台界面
- 13个业务模块完整实现
- TXT + BIN 双格式文件存储
- 5类Bug修复、10类软件测试
- AI方案对比实验

### v2.0 - EasyX图形版 🚧（开发中）
- 基于 C11 + EasyX 图形库
- 图形化用户界面
- 可视化地铁线路图
- 按钮交互操作
